# Found max Original Start Date in order to find last interval days
datasets=[]
IntervalDay = [15,30,45,60,90,180]

for i in final_data['Country Code'].unique():
    for lastdays in IntervalDay:
         
        for_churn_rate = final_data[(final_data['Country Code'] == i)]
        for_churn_rate = final_data[ final_data['Original Start Date'] >= (final_data['Original Start Date'].max() - datetime.timedelta(days=lastdays))]
        for_churn_rate['IntervalDay'] = lastdays
        # Calculated Churn Rate based on Total Churn and newSubs
        for_churn_rate = for_churn_rate.groupby(['Country Code','IntervalDay']).agg({'newSubs':'sum','Total Churn':'sum'})
        for_churn_rate['Churn Rate'] = for_churn_rate['Total Churn'] / for_churn_rate['newSubs']
        for_churn_rate=for_churn_rate.reset_index()
        # Appended Dataset
        for_churn_rate=for_churn_rate[['Country Code','IntervalDay','Churn Rate']]
        for_churn_rate =for_churn_rate.set_index(['Country Code','IntervalDay'])
        datasets.append(for_churn_rate)

App engagement is defined by a set of metrics that measure user interaction with an app. Common engagement metrics can include how many times a user opens the app, the duration of a usage session, and what actions a user takes within an app. As we seen below, first country in the list is U.S.A, and it has remarkable number of engagement. Second row belongs to Japan. Engagement trouble may be because there are too many notifications to customers. There are 3 ways to prevent churn based on engagement in these countries for me. First of all, fewer notifications should be sent to the customer, that is, almost no notifications should be sent, except for important notifications such as updates, customer interests etc. Secondly, Discounts, offers, and cupons plays critical role on customers. Finally, Don't beg to user for positive rating and don't irritate user by pop-ups asking for positive rating on App store. Don't neglect negative feedback/rating on app store and reply to them smartly.

   year = df[column].dt.year
   month = df[column].dt.month
